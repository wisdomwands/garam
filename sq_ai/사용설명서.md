# SQ-AI (Spiritual Intelligence) 시스템 사용 설명서

본 문서는 가람 팀장이 구현한 SQ-AI 시스템의 기본 조작법과 데이터 확장 방법을 안내합니다.

---

## 1. 시스템 실행 방법

### 🔍 1대1 질의응답 (상담) 실행
파이썬 환경에서 `pipeline.py`를 실행하여 시스템의 응답을 확인할 수 있습니다.

```powershell
# sq_ai 디렉토리로 이동
cd d:\페르미아\가람\sq_ai

# 파이프라인 엔진 실행
python pipeline.py
```

- **실행 결과:** 사용자의 질문에 대해 원주님의 로고스 데이터를 검색하고, '질서-균형-조화' 필터를 거친 영적 가이드를 출력합니다.

---

## 2. 지식 데이터 추가 및 업데이트

### 📥 데이터 인명(Ingestion) 스크립트 활용
새로운 텍스트 파일을 `sq_ai/data.json` 형식으로 자동 변환합니다.

1. `d:\페르미아\가람\회로도관련\` 폴더에 텍스트 파일을 넣습니다.
2. `ingest_intro_files.py` 스크립트를 실행합니다.

```powershell
python ingest_intro_files.py
```

---

## 3. 핵심 파일 구성 및 역할

| 파일명 | 역할 | 비고 |
| :--- | :--- | :--- |
| **`models.py`** | 데이터의 '에너지 결' 정의 | Pydantic 스키마 |
| **`prompts.py`** | 최상위 철학적 주파수 설정 | 시스템 가이드라인 |
| **`pipeline.py`** | 질문 분석 및 영적 추론 엔진 | 핵심 RAG 로직 |
| **`data.json`** | 학습된 로고스 아카이브 | 실제 지식 베이스 |

---

## 4. 가람 팀장을 위한 기술 팁 (Advanced)

- **의도 분석 확장:** `pipeline.py` 내의 `analyze_query_intent` 함수에 새로운 키워드나 섹터 매핑 로직을 추가하여 감도를 조절할 수 있습니다.
- **프롬프트 튜닝:** 원주님의 철학이 응답에 더 깊이 배어 나오게 하려면 `prompts.py`의 `CORE_SYSTEM_PROMPT`를 수정하십시오.
